# Preetham's Portfolio - Editorial Brutalist

A premium, conversion-focused interactive portfolio designed to attract high-paying business clients. Built with a raw "Editorial Brutalist" aesthetic, leveraging modern web technologies for a fluid, cinematic experience.

## Tech Stack
- **Framework:** [Next.js 15](https://nextjs.org/) (App Router, Server Components)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Animation & Layout:** [GSAP](https://gsap.com/) (ScrollTrigger) & [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll:** [Lenis](https://lenis.studiofreight.com/)
- **WebGL Background:** [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/) & [Drei](https://github.com/pmndrs/drei)

## Design System
- **Colors:** Carbon (`#030303`), Graphite (`#111111`), Bone White (`#EAEAEA`), Industrial Silver (`#666666`), Acid Yuzu (`#D4FF00`).
- **Typography:** Cabinet Grotesk (Headings), Geist (Body), IBM Plex Mono (Technical/Labels).

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run the Development Server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:** Navigate to [http://localhost:3000](http://localhost:3000) to view the project.
